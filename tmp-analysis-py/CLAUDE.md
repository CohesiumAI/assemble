# Cohesium AI — Agent Workflow System

## Système d'orchestration

Ce projet utilise le système Cohesium AI avec 28 agents spécialisés, des skills partagées, et des workflows prédéfinis.

## Orchestrateur

L'orchestrateur Jarvis est disponible dans `.claude/agents/ORCHESTRATOR.md`.
Il analyse les demandes et séquence automatiquement les agents.

## Commandes disponibles

Les commandes sont définies dans `.claude/commands/`.
